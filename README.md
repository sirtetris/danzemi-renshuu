# Git
簡単なワークフロー：

* `git pull` (リポジトリからプロジェクトの最新のバージョンをゲットする)
* ファイルを変更する
* `git add ◯◯` (変更があるファイルを選ぶ)
* `git commit -m "◯◯"` (変更を保存して、その説明を書く)
* `git push` (リポジトリに送る)
